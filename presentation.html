// Détection sortie plein écran
function handleFullscreenChange() {
    const isNowInFullscreen = !!(document.fullscreenElement 
        || document.webkitFullscreenElement);

    if (!isNowInFullscreen && wasInFullscreen && qcmStarted) {
        fullscreenEscapes++;
        if (fullscreenEscapes === 1) {
            showOverlay(); // 1er avertissement
        } else {
            window.location.href = 'process.php?cheat=1'; // 0/20
        }
    }
    wasInFullscreen = isNowInFullscreen;
}

Sécurité

Sécurité & Bonnes Pratiques

Chaque page est protégée, chaque entrée est validée, chaque session est vérifiée.

🔒

Sessions sécurisées

Vérification session_status() sur chaque page. Redirection si non connecté. Rôle vérifié pour l'admin.

🛡️

Anti-injection SQL

mysqli_real_escape_string() sur toutes les variables utilisateur. Casting explicite des IDs avec (int).

🧼

XSS Protection

htmlspecialchars() systématique sur les données affichées. Jamais d'echo brut.

🚫

Accès direct bloqué

Pages QCM protégées par flags de session. fraud.php inaccessible sans flag de triche.

⏱️

Timer double

Timer côté client (JS) + vérification côté serveur (PHP). Impossible de tricher sur le temps.

🔄

Déconnexion propre

Destruction complète de la session + suppression du cookie côté client.

auth/deconnexion.php 
$_SESSION = array();
if (ini_get("session.use_cookies")) {
    $params = session_get_cookie_params();
    setcookie(session_name(), '', time() - 42000,
        $params["path"], $params["domain"],
        $params["secure"], $params["httponly"]
    );
}
session_destroy();
